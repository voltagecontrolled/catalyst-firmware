## Hardware Panel Reference

### Catalyst Sequencer (Sequencer Mode)

**Inputs (3 total):**
- Top left: Clock In
- Top center: Reset
- Top right: Phase CV

**Outputs (8 total, knob/jack pairs):**
- 1, 2, 3, 4, 5, 6, 7, 8
- Each output has a corresponding encoder knob with a transparent LED cap (color is dynamic, ignore for mapping purposes)

**Named Encoder Knobs (mapped to outputs):**
- Dir. (Direction)
- Phase
- Range
- Transpose
- Start
- Length
- BPM/Clock Div
- Random

**Slider:**
- Phase Scrub (crossfader)

**Buttons (left cluster):**
- Fine
- Copy
- ▶︎‖ / Reset (Play/Reset)
- Glide (Ratchet)

**Pad Grid (2×4):**
- Pages

**Buttons (right cluster):**
- Shift
- Tap Tempo
- Chan. / Quantize (Gate)

- Top right: CV

**Outputs (8 total, knob/jack pairs):**
- 1, 2, 3, 4, 5, 6, 7, 8
- Each output has a corresponding encoder knob with a transparent LED cap (color is dynamic, ignore for mapping purposes)

**Named Encoder Knobs (mapped to outputs):**
- (1–4 unlabeled in panel silkscreen)
- Clock Div
- Random
- (others unlabeled — confirm against manual)

**Slider:**
- Pathway (crossfader)

**Buttons (left cluster):**
- Fine
- Copy
- Play / Rec
- Morph / Range

**Pad Grid (2×4):**
- Scenes

**Buttons (right cluster):**
- Shift
- Add / Delete
- Bank / Quantize (Gate)
